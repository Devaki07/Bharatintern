# House Price Prediction using Linear Regression

## Project Overview

This project was developed as part of my Machine Learning Internship at Bharat Intern. The objective is to predict house prices using Linear Regression based on house features such as living area and number of bedrooms.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The project uses a housing dataset containing information such as:

* Price
* Bedrooms
* Bathrooms
* Square Foot Living Area
* Square Foot Lot
* Floors
* Location Details

## Methodology

1. Loaded and explored the dataset using Pandas.
2. Selected important features:

   * sqft_living
   * bedrooms
3. Split the dataset into training and testing sets.
4. Trained a Linear Regression model.
5. Evaluated the model using Mean Squared Error (MSE).
6. Predicted house prices for new input data.

## Results

* Model: Linear Regression
* Evaluation Metric: Mean Squared Error (MSE)
* Example Prediction:

  * House Size: 2000 sqft
  * Bedrooms: 4
  * Predicted Price: $480,750

## Learning Outcomes

* Data preprocessing and visualization
* Train-test split
* Linear Regression implementation
* Model evaluation
* House price prediction using Machine Learning
